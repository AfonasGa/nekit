Чтобы смотреть ютубчик без впна

Windows

Скачиваем на винду по гайду https://github.com/ValdikSS/GoodbyeDPI/issues/378

либо

Для Google Chrome - chrome://flags

Для Firefox/Waterfox - about:config

Для Microsoft Edge - edge://flags

Для Opera/GX - opera://flags

Напишите в поиск QUIC и включите его, и аналогичным методом отключите Kyber. После этого Ютуб должен заработать (и потом возвращайте всё на места)

Android

скачиваем https://github.com/dovecoteescapee/ByeDPIAndroid/releases, разрешаем подключение впн.
